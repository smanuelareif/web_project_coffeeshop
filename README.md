# Coffee Shop

## Descrição do projeto

Este projeto consiste em uma página web de uma cafeteria, criada para praticar conceitos de HTML e CSS.

Foram utilizadas tecnologias como:

- HTML5 para estruturar a página;
- CSS3 para estilização;
- Metodologia BEM para organização das classes CSS;
- Flexbox para posicionamento dos elementos.

## Melhorias futuras

Algumas melhorias que podem ser implementadas no futuro:

- Adicionar JavaScript para criar um menu interativo
- Criar uma página de pedidos
- Adicionar mais animações
- Melhorar a acessibilidade
- Criar uma versão mobile
